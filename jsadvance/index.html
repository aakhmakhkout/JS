Event Bubbling

The event starts at the target element and then bubbles up through its parents. this is the default behavior of javascript

button → div → body → html → document

Event capturing

Capturing means the event starts from the top of the DOM tree and moves downward toward the actual element you clicked.

document → html → body → div → button
document.querySelector("div").addEventListener( "click", () => console.log("DIV (capturing)"), true // ← enables capturing );
